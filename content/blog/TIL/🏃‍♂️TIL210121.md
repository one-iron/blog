#### 👨🏻‍🎓 오늘 무엇을 배웠니?

```
1. PHP 에서 DynamoDB 연동 작업
		- xde-debug
    우선 XDE 디버깅이 제대로 되지 않았다. xde를 재설치 할 때 php.ini 파일에 해당 설정 내용을 추가 해야했다.

[xdebug]
zend_extension=/Applications/XAMPP/xamppfiles/lib/php/extensions/no-debug-non-zts-20131226/xdebug.so
xdebug.remote_enable=on
xdebug.remote_handler=dbgp
xdebug.remote_host=127.0.0.1
xdebug.remote_port=9000
xdebug.remote_autostart=1

		- Dynamodb
		다이나모 디비 값을 가져오려면 브랜디 내부 함수를 이용해서 가져오는데 이 때 다이나모 json 형태로 주고 받고 해야할 			수 있으므로 디버깅을 잘 하면서 접근하자. `['N']` 같은거 :)  그리고 오타로 에러를 몇 시간동안 못 잡았다.
```
