- apply

  - 함수에 인자를 배열로 넣는 것

    ```javascript
    var person = {
      fullName: function() {
        return this.firstName + " " + this.lastName;
      }
    }
    var person1 = {
      firstName:"John",
      lastName: "Doe"
    }
    var x = person.fullName.apply(person1); 
    document.getElementById("demo").innerHTML = x; 
    ```

    person에 있는 fullname 함수를 호출하고 그 함수 인자에 person1의 객체 키값을 넣고 있다.
